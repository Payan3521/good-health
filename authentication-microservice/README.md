# Authentication Microservice - Good Health

Este directorio contiene el microservicio encargado de la autenticación y gestión de accesos para el proyecto **Good Health**.

## Stack Tecnológico
- **Tecnologías:** Java 21 + Spring Boot 4.0.5
- **Gestor de Dependencias:** Maven 3.9.14
- **Base de Datos:** PostgreSQL (SQL)

## Ejecución
- **Puerto expuesto:** `8080`

### Comandos disponibles

| Acción | Comando |
| :--- | :--- |
| **Run** | `./mvnw spring-boot:run` |
| **Clean** | `./mvnw clean` |
| **Build** | `./mvnw compile` |
| **Clean & Run** | `./mvnw clean spring-boot:run` |
