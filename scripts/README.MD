# Scripts - Good Health

Este directorio está destinado a almacenar todos los scripts utilitarios y de automatización para el proyecto **Good Health**.

## Propósito

Aquí es donde se deben ubicar los diferentes scripts de mantenimiento y configuración, tales como:

- **Seeds de Base de Datos:** Scripts para poblar o inicializar las bases de datos con información de prueba (`seeds`).
- **Migraciones manuales:** Scripts que requieran ejecutarse de manera extraordinaria sobre las bases de datos.
- **Automatización:** Pequeños programas (en Bash, Python, etc.) para tareas repetitivas, como limpieza, generación de tokens manuales, o sincronización de entornos.

> **Nota:** Cada vez que agregues un script importante, considera documentar brevemente en este mismo archivo o en la cabecera del script cómo debe ejecutarse y qué pre-requisitos tiene.
