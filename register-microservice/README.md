# Register Microservice - Good Health

Este directorio contiene el microservicio encargado del registro y la creación inicial de perfiles para nuevos usuarios en el proyecto **Good Health**.

## Stack Tecnológico
- **Tecnologías:** TypeScript 5.7.3 + NestJS 11.0.1
- **Gestor de Dependencias:** npm/npx 11.9.0
- **Base de Datos:** PostgreSQL (SQL)

## Ejecución
- **Puerto expuesto:** `8087`

### Comandos disponibles

| Acción | Comando |
| :--- | :--- |
| **Run** | `npm run start:dev` |
| **Clean** | `rm -rf dist` |
| **Build** | `npm run build` |
| **Clean & Run** | `rm -rf dist && npm run start:dev` |
